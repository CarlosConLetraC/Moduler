# Moduler
Moduler es un motor de scripts multitenencia, útil para poder trabajar con varios "contenedores" ambientados en Lua.

## Sistemas compatibles:
  Debian 13 (Trixie)
  Ubuntu-22.04 Jammy

Para poder iniciar el proyecto
```bash
git clone --recursive https://github.com/CarlosConLetraC/Moduler.git
cd Moduler
chmod +x *.sh
./run.sh
```
